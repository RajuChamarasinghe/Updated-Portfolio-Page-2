About Me

I am proficient in creating webpages using HTML and CSS and have used the Twitter Bootstrap library for styling. I am also proficient in JavaScript and have experience using the ReactJS frontend framework, developing for the backend using NodeJS framework. Some of the JavaScript libraries I have used are Express, Handlebars, jQuery, and Cheerio. Some of the databases I have worked with are MySQL and MongoDB. I also have deployment experience with Heroku and Github.

I not only rely on my coding competency, but also on my significant leadership skills as well. I am a self motivated team player and able to learn new technologies quickly.

I am also a musician and enjoy playing the guitar and keyboard and currently live in Lynnwood, WA with my wife.
